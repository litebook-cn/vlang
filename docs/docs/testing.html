测试(Testing)

// hello.v 
fn hello() string {
    return 'Hello world'
}

// hello_test.v 
fn test_hello() {
    assert hello() == 'Hello world'
}

所有测试函数都必须放在 *_test.v文件中，并以 test_开头。 要运行测试，请执行 v hello_test.v。 要测试整个模块，请运行 v test mymodule。
